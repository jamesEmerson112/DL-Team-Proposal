================================================================================
PHASE 14: TTT — Test-Time Training
================================================================================

Phased TTT with LoRA + global TTT. Multi-stage process:

1. Compile warmup:
  ttt_lora:warming up compile (random tokens, no val data)
  ttt_lora:compile warmup done (104.8s)

2. Timer start + phased TTT header:
  beginning TTT eval timer
  ttt_phased: total_docs:50000 prefix_docs:2000 suffix_docs:48000 num_phases:1 boundaries:[2000]

3. Prefix eval (before any gradient updates):
  ttp: b781/782 bl:2.2188 bb:0.9929 rl:2.2188 rb:0.9929 dl:17258-30330 gd:0
       ^^^                                                               ^^^
       batch progress                                                   gd:0 = no gradient updates yet

4. Phase complete + global TTT chunks:
  ttpp: phase:1/1 pd:2448 gd:2000 t:174.2s
  tttg: c1/250 lr:0.001000 t:0.3s
  tttg: c2/250 lr:0.001000 t:0.4s
  ...
  tttg: c249/250 lr:0.000000 t:21.0s

5. Phase result:
  ttpr: phase:1/1 t:197.2s

6. Suffix eval (with gradient updates, gd:1):
  ttp: b742/782 bl:2.4039 bb:0.9994 rl:2.2393 rb:0.9937 dl:2730-2762 gd:1
  ttp: b731/782 bl:2.4194 bb:0.9956 rl:2.2552 rb:0.9938 dl:2377-2414 gd:1
  ...
  ttp: b7/782 bl:2.8355 bb:1.1623 rl:2.3647 rb:1.0034 dl:101-103 gd:1

7. Final result:
  quantized_ttt_phased val_loss:2.40073153 val_bpb:1.00692894 eval_time:352953ms
  total_eval_time:353.0s

--- ttp field definitions ---
  ttp: bN/M bl:X.XXXX bb:X.XXXX rl:X.XXXX rb:X.XXXX dl:N-N gd:N
       |     |         |         |         |         |      |
       |     |         |         |         |         |      gradient updates done (0=prefix, 1=suffix)
       |     |         |         |         |         doc length range in batch
       |     |         |         |         running BPB (cumulative)
       |     |         |         running loss (cumulative)
       |     |         batch BPB (this batch only)
       |     batch loss (this batch only)
       batch progress (remaining/total, counts DOWN)

--- tttg field definitions ---
  tttg: cN/M lr:X.XXXXXX t:X.Xs
        |     |            |
        |     |            wall time elapsed
        |     current learning rate (cosine decay)
        chunk progress

--- ttpp/ttpr field definitions ---
  ttpp: phase:N/M pd:N gd:N t:X.Xs      (phase prefix complete)
        |         |    |    |
        |         |    |    time for prefix eval
        |         |    gradient docs processed
        |         prefix docs processed
        phase progress

  ttpr: phase:N/M t:X.Xs                 (phase global TTT complete)
        |         |
        |         time for global TTT training
        phase progress

  12. Serialized model + Code size (uncompressed) + Code size (compressed)

  14. diagnostic quantized val_loss/bpb/eval_time

  16. beginning TTT eval timer + ttt_phased header
  17. ttp prefix (gd:0) → ttpp → tttg chunks → ttpr
  18. ttp suffix (gd:1, ~90 lines)
  19. quantized_ttt_phased val_loss/bpb/eval_time
